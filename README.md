# Guitar-Jiro [UCSP]
Guitar Jiro es un proyecto donde tenemos interacción con barras, que se van aproximando al compás de la música.

## Required Software
 * **Unity3d 2018**
 * **Leap Motion SDK V"**

## Required Dependencies
 * QuickTime

## Compile 
```
- Open Unity3d
- Seleccionar escena Menu
- Comenzar a Jugar
```
